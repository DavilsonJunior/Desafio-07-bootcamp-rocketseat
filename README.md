## :rocket: Executando

```bash
# Clone o Repositório
$ git clone https://github.com/DavilsonJunior/Desafio-07-bootcamp-rocketseat
```

```bash
# Acesse a pasta do projeto e baixe as dependências
$ yarn
```

```bash
# Para verificar os testes use:
$ yarn test
```

```bash
# Rodando o server
$ yarn start
```
